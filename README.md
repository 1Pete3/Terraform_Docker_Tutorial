# Terraform Docker Tutorial

## Overview
Learning Terraform using the Docker provider to manage 
containers and images as infrastructure as code.

## What it provisions
- Docker image (nginx)
- Docker container on port 8080

## Prerequisites
- Terraform installed
- Docker installed

## How to run
# Initialise Terraform
terraform init

# Preview changes
terraform plan

# Apply changes
terraform apply

# Clean up
terraform destroy

## What I learned
- Terraform basic syntax and workflow
- Resource dependencies
- State management
- terraform validate and plan
